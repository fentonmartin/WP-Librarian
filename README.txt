WP-Librarian
============

WP-Librarian is a WordPress plugin that allows any WordPress site to be used to display and manage an extensive library of media. Items can be given extensive information including Author/ISBN/Donor/Category/Tags/Condition and loans to members of the Library are tracked within the plugin, so the librarians can easily monitor which books are due back and when. Additional fining funcionality allows Members to be charged for items returned late.

WP-Librarian is currently under development with a planned release in early Setember 2014.
